The Iris Flower Prediction project is a machine learning project that predicts the species of an Iris flower based on its features. The project uses the classic **Iris dataset**, which contains measurements of sepal length, sepal width, petal length, and petal width for three species: **Setosa, Versicolor, and Virginica**.

The goal of this project is to build a predictive model that can classify iris flowers accurately based on these features.
